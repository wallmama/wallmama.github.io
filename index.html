翻墙/科学上网/VPN推荐，商业/自建机场

到2025年底，中国对VPN的封锁力度非常大，“能用"的标准已经变成了"能连上且速度尚可”。曾经很多老牌的大型商业VPN（如ExpressVPN、NordVPN）在中国的连接成功率已经非常不稳定，不建议作为首选。

以下是目前（2025年末）中国境内实测最稳妥的几类选择，按推荐程度排序：

第一梯队：目前最稳的商业VPN推荐

如果你不想折腾技术，只想要一个下载就能用的App，目前这两个是最主流的选择：

Astrill VPN

官网

  • 现状： 外企员工和在中国的老外最常用的高端选择，被称为"VPN界的爱马仕"。
  • 优点： 极度稳定： 在两会、敏感日等封锁严重时期，它往往是唯一还能连上的VPN。
    • 协议强大： 使用其专有的 StealthVPN 或 OpenWeb 协议，过墙能力极强。
    • 功能全： 支持5台设备，且支持路由器设置。
  • 缺点：
    • 极贵： 价格约为每月30美元（约合200+人民币），如果不买一年几乎无法接受。
    • 体验一般： App界面非常古老，甚至有点像Windows 98时代的软件，但这不妨碍它好用。
    • 无退款： 不支持退款。

LetsVPN (快连 VPN)

官网

  • 现状： 目前在中国普及度最高、连接最稳的"傻瓜式"VPN。
  • **优点：专攻中国市场： 它的协议（私有ECC协议）专门为了过墙设计，连接速度极快（经常是秒连）。
    • 操作简单： 界面只有一个按钮，甚至不需要选服务器，它会自动分配。
    • 价格适中： 比Astrill便宜很多。
  • 缺点/风险：
    • 设备限制严： 标准账号只能同时在线2台设备，而且抓得很严（不像其他家可以随意踢人，它需要手动解绑）。
    • 隐私争议： 公司背景不如国际大厂透明，且有用户反馈进行BT下载或大流量传输时会被封号（它更适合查资料、看视频，不适合挂机下载）。
    • 退款困难： 并没有真正的"30天无理由退款"，一旦买了很难退钱。

第二梯队：备用/特定配置可翻墙

这些国际大厂名气很大，但在中国的表现时好时坏，或者需要复杂的特定设置。不要只买这就去中国，风险很大。

  • ExpressVPN： 曾经的王者，现在很吃力。官方推荐使用"Automatic"协议，但经常需要不断切换服务器才能碰运气连上。只能作为备用。
  • NordVPN / Surfshark： 只有在手动配置（如连接特定的"Obfuscated Servers"混淆服务器）或使用OpenVPN手动配置文件时才偶尔能用，App直连成功率极低。
  • Mullvad： 以隐私著称，但直连中国几乎不可用，除非你有技术能力配置Shadowsocks桥接。

第三梯队：技术流/自建/机场（最推荐长期用户科学上网）

如果你懂一点技术，或者愿意折腾，这才是目前在中国最主流、性价比最高的上网方式。这通常不叫"VPN"，而叫"代理"（Proxy）。如果不懂技术，不妨直接买商业机场。

  • 机场 (Shadowsocks/V2Ray/Trojan)： * 原理： 你购买服务商提供的订阅链接，导入到客户端（如 Clash, Shadowrocket, v2rayNG）中使用。
    • 优点： 速度通常比VPN快得多，价格便宜（每月10-30元人民币），分流功能强大（访问国内网站不走流量，访问国外才走）。
    • 缺点： 机场跑路风险高，需要自己找可靠的供应商；iOS上安装客户端（Shadowrocket）需要非中国区的Apple ID。
  • 自建节点 (VPS + Hysteria 2 / Xray)：
    • 现状： 目前 Hysteria 2 协议是过墙的新秀，基于UDP，速度极快且难以被封锁。
    • 适合： 有Linux基础，想要独享IP，不信任第三方服务的人。

核心警告与建议

  1. 必须在入境前下载安装： 一旦进入中国，Google Play、App Store（中国区）、以及上述所有VPN的官网全部无法访问。如果你没有提前装好，你将陷入"没有VPN就下载不了VPN"的死循环。
  2. 永远准备两个方案： 不要把赌注全压在一个App上。例如：买一个月Astrill作为主力，手机里再装一个免费的（如Windscribe，虽然只有10G流量且慢，但能用来救急）。
  3. 关于免费VPN： 几乎所有标榜"免费"的VPN在中国都无法使用，或者极其不安全（会记录你的隐私）。能用的免费梯子通常都有巨大的流量限制或速度限制。
  4. 谨言慎行： 虽然很多人用VPN，但根据中国法律（《计算机信息网络国际联网安全保护管理办法》），擅自建立、使用非法定信道进行国际联网是违规的。仅用于工作、学习和娱乐，严禁在墙外发表敏感政治言论，这是保护你自己的底线。

自建节点 (VPS + Hysteria 2 / Xray)，怎么做？

这是一个非常硬核但也最一劳永逸的选择。自建节点虽然门槛高，但一旦搭建成功，你将拥有独享IP（不会被其他人牵连封号）和极高的性价比。

针对目前的网络环境，我强烈建议你采用 “一键脚本” 方式搭建。手动配置配置文件（Config file）极其容易出错且难以维护。

以下是目前最主流、成功率最高的搭建流程：

第一步：准备 VPS (服务器)

你需要一台位于中国境外的 Linux 服务器。

  • 系统要求： 推荐安装 Debian 11/12 或 Ubuntu 22.04（Debian 占用资源更少，更稳）。
  • 购买建议：
    • 土豪/追求极速： BandwagonHost (搬瓦工) 的 CN2 GIA 线路。
    • 性价比： RackNerd, CloudCone (便宜，一年只需10-20美元，但晚高峰可能会慢)。
    • 避坑： 尽量避开 AWS、Google Cloud、Azure 的热门网段，这些IP段经常被整段屏蔽。

第二步：连接 SSH

购买 VPS 后，你会获得 IP地址、用户名 (root) 和 密码。你需要通过 SSH 工具连接服务器。

  • Windows: 使用 PowerShell 或下载 Putty / FinalShell。
  • Mac: 直接打开“终端” (Terminal)。

命令格式：

ssh root@你的服务器IP
# 例如：ssh root@192.168.1.1
# 输入后回车，然后输入密码（输入密码时屏幕不会显示字符），再回车。

第三步：选择协议与安装 (核心步骤)

目前有两个流派，请根据你的情况选择：

方案 A：VLESS-Reality (Xray内核) —— 推荐新手

  • 优点： 不需要域名（这是最大的优势），极度隐蔽，它会将你的流量伪装成访问 Yahoo、Apple 或 Microsoft 等大网站。
  • 缺点： 速度上限不如 Hysteria 2 暴力。
  • 适用： 第一次尝试自建的人，或者只想安安静静上网的人。

方案 B：Hysteria 2 (歇斯底里协议) —— 推荐速度党

  • 优点： 基于 UDP 协议，拥有暴力的拥塞控制，在垃圾线路上也能跑满宽带（例如用便宜的 VPS 跑 4K 视频）。
  • 缺点： 部分地区的运营商（ISP）会针对 UDP 流量进行限速（QoS），导致连不上或断流；通常建议搭配域名使用（虽然也可以用自签证书，但部分客户端支持不好）。

执行一键安装脚本

为了简化操作，我们使用社区维护的开源整合脚本（集成了 Xray, Hysteria 2, Sing-box 等）。 (注：以下命令来源于 GitHub 知名开源项目，如 Misaka 或 233boy)

在 SSH 窗口中复制粘贴以下任意一条命令并回车（如果第一条失效，试第二条）：

推荐脚本 1 (vasma):

wget -P /root -N --no-check-certificate "https://raw.githubusercontent.com/vasma-coding/hysteria2-script/main/hysteria2.sh" && chmod 700 /root/hysteria2.sh && /root/hysteria2.sh

推荐脚本 2 (甬哥/Yongge - 功能非常全):

bash <(curl -Ls https://raw.githubusercontent.com/yonggekkk/sing-box-yg/main/sb.sh)

安装过程引导：

  1. 脚本运行后会出现中文菜单。
  2. 选择 “安装 Hysteria 2” 或 “安装 VLESS-Reality”。
  3. 端口 (Port)： 默认即可（通常是 443 或 8443）。
  4. 证书：
    • 如果是 Reality：脚本会自动生成，不需要你做任何事。
    • 如果是 Hysteria 2：脚本通常会问你有无域名。选 “无域名/自签证书” (Self-signed) 最快，但客户端需要开启 “允许不安全连接” (Allow Insecure)。
  5. 生成链接： 安装完成后，屏幕上会显示一长串以 vless:// 或 hysteria2:// 开头的链接。把这个复制保存下来，这就是你的"钥匙"。

第四步：客户端配置

拿到链接后，你需要下载支持这些新协议的客户端。普通的旧版 VPN 软件不支持这些。

平台 推荐客户端 操作方法
iOS Shadowrocket ($2.99, 需美区ID) 打开 App -> 左上角扫描二维码，或直接复制链接后 App 会自动提示导入。
Android v2rayNG 或 Nekobox (免费, GitHub下载) 打开 App -> 右上角 “+” -> “从剪贴板导入”。
Windows v2rayN 或 Clash Verge Rev 推荐 v2rayN。添加服务器 -> 从剪贴板导入。
Mac V2Box 或 Clash Verge Rev 同上。

第五步：避坑指南 (重要！)

  1. 防火墙端口： 如果你买了 VPS，脚本也跑通了，但就是连不上，99% 是因为 VPS 服务商的后台防火墙没开端口。你需要去 VPS 网页控制台 (Security Group / Firewall) 放行你刚才设置的端口 (TCP 和 UDP 都要放行)。
  2. IP 被墙检测： 如果某天突然连不上了，去 ping.pe 这个网站，输入你的 VPS IP。
    • 如果中国地区全是红色 (Time out)，而国外全是绿色，恭喜你，你的 IP 被墙了。
    • 解决方法： 花钱换 IP，或者套 Cloudflare CDN（会减速，且只适用于 WebSocket 模式，不详述，建议直接换 IP）。
  3. Hysteria 2 的设置技巧： 如果使用 Hysteria 2，在客户端设置里，把 “跳过证书验证” (Allow Insecure) 打开，否则自签证书会导致连接失败。

总结建议

如果你从未接触过 Linux：

  1. 先买一个便宜的 RackNerd VPS。
  2. 使用 VLESS-Reality 协议（因为不需要域名，配置最少）。
  3. 成功连上后，如果觉得速度不够快，再在同一个 VPS 上重装脚本尝试 Hysteria 2。

哪些vps厂商的ip可用率高？

在2025年的环境下，没有任何一家厂商能保证分配给你的IP 100% 是没被墙的。

所谓的“可用率高”，现在的定义其实分为两类：

  1. “钞能力”类： 价格贵，线路极好，通常保证IP可用（或者换IP政策宽松）。
  2. “无限重开”类（推荐）： 支持按小时计费。开一个服务器如果不通，立刻删掉（只扣几分钱），再开一个新的，直到刷出一个能用的IP为止。

以下是根据不同策略推荐的厂商清单：

策略一：按小时计费（“刷”IP流）—— 首选推荐

这是最经济实惠、风险最低的策略。你不需要赌运气，也不用担心买了一年不能退款。

1. Vultr

  • 特点： 全球机房极多，支持支付宝。
  • 策略：
    • 充值 $10 左右。
    • 创建一个最便宜的实例（比如 $5/月，合 $0.007/小时）。
    • 核心操作： 拿到IP立刻去 ping.pe 测试。全红？-> Destroy (销毁) -> 再重新 Deploy (部署)。
    • 通常尝试 2-3 次就能刷到一个干净的 IP。
  • 推荐机房： 西雅图 (Seattle)、硅谷 (Silicon Valley)。尽量避开东京，那里是被墙重灾区。

2. DigitalOcean (DO) / Linode (现 Akamai)

  • 特点： 老牌大厂，IP池子巨大，也是按小时计费。
  • 注意： 注册时风控较严，尽量用真实信息，可能需要信用卡或 PayPal 激活。
  • 优势： 它们的新加坡或旧金山机房，配合 Hysteria 2 协议，速度往往有惊喜。

3. AWS (亚马逊云) / GCP (谷歌云)

  • 特点： 拥有“换IP”神器（Elastic IP）。
  • 策略： 你不需要删服务器，只需要在后台解绑旧IP，申请新IP，再绑定。
  • 风险： 计费极其复杂，流量费贵。虽然第一年有免费额度（Free Tier），但一旦超额（或者被攻击），信用卡会被扣爆。新手慎入，除非你很懂云服务器计费逻辑。

策略二：“钞能力”类（线路王/懒人流）

如果你预算充足（预算 > $50/年），想要最好的体验，不想折腾刷IP。

1. BandwagonHost (搬瓦工)

  • 地位： 曾经的“传家宝”，现在的“高富帅”。
  • IP质量： 它的 CN2 GIA 线路是公认最稳、延迟最低的。
  • 保障： 虽然价格贵（通常 $49/年起，好线路甚至 $100+），但它有一个服务：如果你买来发现IP是被墙的，或者用着用着被墙了，可以在后台付费换IP（几美元一次），或者某些高端套餐免费换。
  • 适合： 追求极致稳定，不差钱的用户。

2. DMIT

  • 特点： 近几年的新贵，线路质量极高。
  • 政策： 它的高端系列通常承诺 IP 交付时是可用的。如果在购买后一定时间内IP被墙，部分套餐提供更换保障（具体需看当时条款）。
  • 适合： 想要媲美专线速度的用户。

策略三：性价比“赌”IP（RackNerd, CloudCone）

这就是我们上一轮提到的类型。

  • 现状： 它们非常便宜（$10-$15/年），所以吸引了大量国人购买，导致IP池非常脏。
  • 风险： 购买后大概率（50%以上）遇到被墙IP。
  • 补救： RackNerd 换一次 IP 需要 $3。哪怕你加上这 $3，总价依然比 Vultr 便宜。所以它依然是性价比之王，只要你做好了“可能需要花钱换一次IP”的心理准备。

总结与建议

  • 如果你想要绝对省心 + 极速： 去买 BandwagonHost (搬瓦工) 的 CN2 GIA 套餐。
  • 如果你有技术基础 + 想省钱 + 不怕麻烦： 用 Vultr 或 DigitalOcean，利用“按小时计费”机制，手动刷出一个干净的 IP，然后一直用。
  • 如果你只要最便宜 + 愿意冒点险： 坚持买 RackNerd。
    • 小技巧： 购买 RackNerd 时，在订单备注（Notes）里用英文写一句：“Please allocate a clean IP usable in China if possible.” 虽然不一定有用，但有时候客服看到了会人工帮你挑一个。
